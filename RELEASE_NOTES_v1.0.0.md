# Release v1.0.0

## Highlights

- Local AI agent built on top of Ollama
- Arabic-first web interface
- Model routing for chat, code, and vision flows
- Memory, retrieval, and workflow tracking
- Full project builder with target-directory support
- Project execution pipeline for run and smoke-test flows
- Built-in templates for Python, Flask, FastAPI, Node, React, and static web starters
- Docker support, tests, and GitHub Actions CI

## Included Capabilities

- Build a project scaffold from a plain-language description
- Generate `project_spec.json` and `NEXT_STEPS.md`
- Save conversations, settings, telemetry, and workflow runs
- Export and import conversations as JSON
- Apply built-in templates from the UI
- Execute project pipelines and collect first-pass remediation notes

## Validation

- Automated test suite passing locally
- Git repository initialized and connected to GitHub
- Main branch updated with the complete project state
